# vue2-sandbox
Vue 2.6 with TailwindCSS, Portal Vue, Composition API Sandbox for my own learning purposes.. ;)

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
